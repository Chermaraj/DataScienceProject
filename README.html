Application Setup Guide

This document guides developers to setup our applciation source code and run it from their own machines

Step 1: Clone using github repository

In this step you can download the source code to your local by cloning the repository.

  1. Copy the repository URl.

    https://github.com/Chermaraj/DataScienceProject

  2. Input the repository URL and local path as shown in image

  3. Enter the project details when prompted. This creates a new project.

Step 2: Install Dependencies

In this step you have to install the dependencies that required to run the application

  1. Right click on env variable and select install from requirements.txt

    This will install all the dependencies to your local machine

Step 3: Download and place data

In this step you have to download the data and place at corresponding folder

  1. Download the data.zip from below URL.

    https://drive.google.com/file/d/1CPOh7HW2qnjsxIYfNnnPUd7DETcYWJK2/view?usp=sharing

  2. Unzip the data.zip under 'DataScienceProject\data' directory

Step 4: Check Hardware requirement

Plese make sure your machine has NVIDIA driver to run CUDA latest version

Step 5: Run the program

After installing all the requirements and dependencies, now we can run the applciation

  1. Opne the command terminal at root directory and input the below command

  2. python manage.py runserver

Step 6: Copy application URL and run it

When there is no complie time error, the command would produce a URL. Please copy the URL and paste it in browser. You can see the webiste homescreen now. Thank you
